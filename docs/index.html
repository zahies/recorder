ESP32 Sound Recorder

ICS43434 I2S Microphone — WiFi Stream to Phone

Install Firmware

Connect your Heltec WiFi LoRa 32 via USB-C, then click install.

Your browser doesn't support Web Serial.
Use Chrome or Edge on desktop. Web Serial is not allowed in this context.

How to Use

  1. After flashing, connect your phone to WiFi: ESP32-Recorder (password: za12xs43)
  2. Open http://192.168.4.1 in your phone's browser
  3. Tap Record to start recording
  4. Tap Stop to stop
  5. Tap Download WAV to save the file

Requires Chrome or Edge. Works on ESP32-S3 (Heltec WiFi LoRa 32 V3/V4).
